# super-pano-editor
Logiciel graphique de configuration pour un prototype de trépied pivotant.
Réalisé dans le cadre d'un projet scolaire.

Pour executer le programme vous avez besoin des extensions suivantes :
<ul>
<li>Tkinter</li>
<li>pickle</li>
</ul>
